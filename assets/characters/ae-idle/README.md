# AE Idle Character

Cartella per il personaggio Lottie esportato da After Effects.

File attesi:

- `idle.json` (Bodymovin/Lottie)
- opzionale `preview.png` per fallback statico

Di default il template usa:

- `idle.json` come animazione
- `assets/characters/boks_black/placeholder.png` come preview fallback

Se vuoi usare `preview.png`, aggiorna `previewSrc` in:

- `assets/animations/characters/ae-idle/manifest.js`
